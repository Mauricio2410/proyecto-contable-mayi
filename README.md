## 🛠️ Instrucciones de Ejecución

Si deseas ejecutar este proyecto localmente, sigue estos pasos:

1. **Clonar el repositorio:** `git clone https://github.com/TU_USUARIO/proyecto-contable-mayi.git`
2. **Entrar a la carpeta:** `cd proyecto-contable-mayi`
3. **Instalar dependencias:** `npm install` (Esto descargará la carpeta node_modules que no subimos por espacio).
4. **Iniciar la aplicación:** `npm start`
